## deepface 사용법
* Keras와 TensorFlow를 바탕으로 만든 얼굴 인식 라이브러리입니다. 
* ```pip install deepface```로 설치합니다.
```
pip install deepface
```

* https://www.tensorflow.org/install/pip?hl=ko#virtual-environment-install 에서 파이썬 버전을 설정합니다.
